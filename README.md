# AegisOS 🛡️

Bienvenido al repositorio oficial de **AegisOS**, un sistema operativo diseñado para ofrecer una experiencia personalizada, eficiente y privada.

## ⚠️ AVISO IMPORTANTE (DESCARGO DE RESPONSABILIDAD)

> **POR FAVOR, LEE ESTO ANTES DE INSTALAR:**
> Este sistema operativo se encuentra actualmente en **fase experimental de desarrollo** y **NO ha sido probado en hardware real** todavía. 

* **Uso bajo tu propio riesgo:** Al descargar e instalar este software, aceptas que el desarrollador no se hace responsable de pérdidas de datos, fallos de hardware, bloqueos del sistema (*bricks*) ni de cualquier otro daño causado en tu ordenador.
* **Recomendación estricta:** Se recomienda encarecidamente probar AegisOS única y exclusivamente en entornos controlados, como una **Máquina Virtual** (VirtualBox, VMware, QEMU, etc.), antes de intentar cualquier instalación real.

## 🚀 ¿Cómo probarlo?

1. Dirígete a la sección de **Releases** (Lanzamientos) a la derecha de esta página.
2. Descarga la última versión de la imagen oficial (archivo `.iso`).
3. Monta la imagen en tu software de máquina virtual preferido y arráncala.
